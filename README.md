# Habit Grow

if-thenルールで習慣を登録し、日々の達成と週次/隔週レビューで少しずつ育てるブラウザアプリです。

## 現在の状態

- ビルド不要の静的HTML/CSS/JavaScript
- localStorage保存
- GitHub Pagesでそのまま公開可能
- Supabase連携前のMVP

## 起動方法

ファイルを直接開く:

```text
index.html
```

ローカルサーバーで開く:

```bash
python3 -m http.server 4173
```

その後、ブラウザで以下を開きます。

```text
http://localhost:4173
```

## 主な機能

- 習慣登録
- if-thenルール
- 単発習慣 / ルーティン習慣
- 既存習慣のコピー作成
- ステップ追加式のルーティン編集
- ステップごとの集計開始日
- ルーティンのステップ別チェック
- `? 天気` / `- 晴れ: 散歩する` のような分岐ステップ
- レビューで開始率・完了率・ステップ別達成率を表示
- 分岐ごとの選択/完了状況を表示
- ステップ間のつまずきポイントを可視化
- 平日・土日・祝日・土日祝・祝日除く平日の頻度設定
- 祝日リストの編集
- 最小達成条件
- タグ付きメリットライブラリ
- およその実施時間順の今日リストと、記録済み項目の後方移動
- 今日の達成/未達/スキップ記録
- JSONバックアップの書き出し/読み込み
- 1週間/2週間/3週間/30日/カスタム周期レビュー
- 達成率と体感難易度に応じた調整提案
